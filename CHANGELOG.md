# Changelog

각 릴리스의 사용자-체감 변경 사항. **릴리스 노트와 인앱 업데이트 안내문이 여기서 생성된다** —
`release.sh`(macOS) / `release.ps1`(Windows)가 배포할 버전의 섹션을 추출해 노트 맨 앞에 넣고,
해당 버전 섹션이 없으면 배포를 중단한다.

형식: `## <버전> (<플랫폼>) - <YYYY-MM-DD>` 제목 아래 `- ` 불릿. 버전은 OS별 독립 트랙이므로
플랫폼 표기를 생략하지 말 것.

## 2.3.0 (macOS) - 2026-07-11
- 자폭·대폭발이 진짜 폭발처럼 나옵니다 — 시전자 자리에서 큰 주황색 폭발이 터지고, 화면 플래시와
  흔들림이 함께 나오며, 더 이상 상대에게 돌진하지 않습니다
- 자폭으로 양쪽이 함께 쓰러졌을 때, 기절한 야생 포켓몬이 사라지지 않고 화면을 계속
  돌아다니던 버그를 수정했습니다

## 2.2.1 (Windows) - 2026-07-11
- 설치 위치를 직접 선택할 수 있습니다
- 설치 중 바탕화면 아이콘 생성 여부를 선택할 수 있습니다
- 일부 환경에서 시작 메뉴 바로가기 이름이 깨지던 문제를 수정했습니다

## 2.2.0 (Windows) - 2026-07-11
- 홈페이지 다운로드 버튼과 인앱 업데이트가 OS별 최신 버전을 정확히 찾도록 개선했습니다
  (한쪽 OS만 새 버전을 올려도 다른 OS에 잘못된 업데이트가 뜨지 않습니다)
- 트레이 메뉴에 개발용 디버그 서브메뉴가 추가됐습니다

## 2.1.0 (macOS · Windows) - 2026-07-11
- Windows에서도 라이징 모드를 지원합니다 — 배틀, 아이템, 진화, 프롬프트 UI
- Windows 다중 모니터 / 가상 데스크톱 환경에서 오버레이가 안정화됐습니다
- 같은 시드의 배틀이 macOS와 Windows에서 동일하게 재생되도록 검증을 추가했습니다

## 2.0.0 (macOS · Windows) - 2026-07-11
- 첫 Windows 버전 출시 (인스톨러 + 무설치 zip)
- 홈페이지에 macOS / Windows 플랫폼별 다운로드 버튼이 생겼습니다
